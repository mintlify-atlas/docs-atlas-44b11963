This page walks you through the essentials of Nextcloud Text: opening a document, formatting content, switching views, sharing with collaborators, and using keyboard shortcuts. You can follow these steps in any Nextcloud 17+ instance with the Text app enabled.

## Open or create a document

<Steps>
  <Step title="Open an existing file">
    In **Nextcloud Files**, click any `.md` or `.txt` file. It opens directly in the Nextcloud Text editor.
  </Step>
  <Step title="Create a new document">
    Click the **+** (New) button in the Files toolbar. Select **New document** to create a new `.md` file. Give it a name and press Enter — the editor opens immediately.
  </Step>
</Steps>

## Edit your document

The editor opens in rich-text (WYSIWYG) mode by default. You can type freely and apply formatting using the toolbar at the top of the editor.

### Using the toolbar

The toolbar provides quick access to the most common formatting options:

- **Headings** — Select a paragraph and choose H1–H6 from the heading dropdown.
- **Bold** — `Ctrl+B` / `Cmd+B`, or click the **B** button.
- **Italic** — `Ctrl+I` / `Cmd+I`, or click the **I** button.
- **Lists** — Click the ordered or unordered list buttons to start a list. Press Tab to indent a list item.
- **Code block** — Click the code block button to insert a fenced code block with syntax highlighting.
- **Blockquote** — Click the quote button to wrap a paragraph in a blockquote.
- **Math** — Click the math button to insert a LaTeX formula block. Inline math can be typed directly with `$formula$`.

### Switching between rich-text and source view

To view or edit the raw Markdown source, click the **Source** toggle in the editor toolbar. Switch back to rich-text mode by clicking **Rich text**.

<Note>
  Switching views does not change the file content — the document is always saved as Markdown regardless of which view you use.
</Note>

## Share a document and collaborate

<Steps>
  <Step title="Share the file">
    In Nextcloud Files, right-click the document (or open the file details panel) and click **Share**. Enter a Nextcloud username, group, or generate a public share link.
  </Step>
  <Step title="Invite collaborators to edit">
    When sharing with a specific user, make sure the **Can edit** permission is enabled. Users with edit access can open the document and type at the same time as you.
  </Step>
  <Step title="Open the document together">
    Both you and your collaborator open the document in Nextcloud Files. You will see each other's presence indicators — colored cursors with names — appear in real time as soon as a second person opens the file.
  </Step>
  <Step title="Edit simultaneously">
    Type in any part of the document. Changes from all active collaborators appear instantly. Yjs CRDT conflict resolution ensures no edits are lost, even if two people type in the same location at the same time.
  </Step>
</Steps>

<Tip>
  You can share a document with anyone, even people who do not have a Nextcloud account, using a public share link. Guests can view or edit based on the permissions you set.
</Tip>

## Mention a collaborator

Type `@` followed by a username anywhere in the document to mention a Nextcloud user. A suggestion popover appears — select the user to insert the mention. The mentioned user receives a Nextcloud notification.

## Keyboard shortcuts

| Action | Shortcut |
|---|---|
| Bold | `Ctrl+B` / `Cmd+B` |
| Italic | `Ctrl+I` / `Cmd+I` |
| Underline | `Ctrl+U` / `Cmd+U` |
| Undo | `Ctrl+Z` / `Cmd+Z` |
| Redo | `Ctrl+Shift+Z` / `Cmd+Shift+Z` |
| Heading 1 | `Ctrl+Alt+1` / `Cmd+Alt+1` |
| Heading 2 | `Ctrl+Alt+2` / `Cmd+Alt+2` |
| Inline code | `Ctrl+E` / `Cmd+E` |
| Soft line break (inside math block) | `Shift+Enter` |
