Nextcloud Text is the default document editor for Nextcloud, enabling real-time collaborative editing of Markdown files with a clean WYSIWYG interface. Built on [Tiptap](https://tiptap.dev) and [ProseMirror](https://prosemirror.net), it combines the openness of Markdown with the ease of a rich-text editor — so your files stay portable while your editing experience stays polished.

## Get started

<Steps>
  <Step title="Enable the app">
    Nextcloud Text comes pre-installed with Nextcloud. Enable it from **Apps** → search "Text" → **Enable**.
  </Step>
  <Step title="Open a document">
    In Nextcloud Files, click any `.md` or `.txt` file to open it in the editor. Or create a new file using the **+** menu.
  </Step>
  <Step title="Share and collaborate">
    Click **Share** on any file to invite collaborators. Multiple users can edit simultaneously with live presence indicators.
  </Step>
  <Step title="Integrate with your app">
    Use `window.OCA.Text.createEditor()` to embed the editor inside your own Nextcloud app. See the [Integration guide](/integration/embed-editor).
  </Step>
</Steps>
